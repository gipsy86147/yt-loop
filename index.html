Static player

Loop any YouTube segment.

Enter a YouTube video ID and one or more time ranges. Open this file in any browser, press play, and the selected segments will repeat in order.

Times can be entered as seconds or hh:mm:ss. The browser still needs internet access for YouTube itself.

Direct file:// playback may fail with YouTube error 153. If that happens, run the included open-local.command launcher instead.
