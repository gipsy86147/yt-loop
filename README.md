# yt-loop

Static HTML app for looping one YouTube video across one or more time segments.

## Local use

Open `open-local.command`, or run:

```zsh
python3 -m http.server 18765
```

Then visit:

`http://127.0.0.1:18765/index.html`

Direct `file://` loading may fail with YouTube error `153`.

## GitHub Pages

This repo is ready to publish as a static site with GitHub Pages from the branch root.
